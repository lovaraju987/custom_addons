Product Low Stock Notification

The Product Low Stock Notification is an Odoo application that helps you keep track of your inventory levels and ensures that you never run out of stock. This application works by sending you an email notification when the stock level of a particular product falls below a pre-defined threshold. This way, you can take immediate action and reorder the product before it runs out completely.

With this application, you can easily configure the low stock notification settings for products, set the minimum stock level threshold, and specify users/groups where the notifications should be sent. Once the notification threshold is reached, the application will automatically send an email to the specified users/groups, alerting you of the low stock level and providing all the necessary details about the product.

This application is particularly useful for businesses that deal with a large inventory of products, as it can be difficult to keep track of stock levels manually. By automating the process of monitoring inventory levels and sending notifications, this application helps you save time, reduce the risk of stockouts, and improve customer satisfaction by ensuring that products are always available when needed.

Module Features
  • Configuration for setup minimum stock level threshold like Global, Individual Product, Reorder rule and Products Category.
  • Selection for outgoing email server
  • Options to select users or groups to send emails
  • Email Notification send to specified users or groups
  • 50 Days Free Support

Selection to set minimum stock level threshold

If you select Global option then you can see minimum level threshold field in settings.

If you select Individual Product option then you can see minimum level threshold field in product variant.

If you select Product Category option then you can see minimum level threshold field in product category.

If you select Reorder Rule option then you can see minimum level threshold field in Reorder Rule.

Configuration of who should receive emails.

You can choose direct users here who will receive low-stock notification email

You can choose user groups also. Whatever group you select here all the users who have been assigned that group will receive email notifications about product low stock

You can also choose outgoing emails here.
